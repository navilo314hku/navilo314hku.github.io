Project Overview

Deep-Learning-Based motion sensor writing glove allows the wearer to draw a number in the air using finger. Motion data is collected during the swing of the finger, transferred to the deep learning model for predicting the number.

Hardware

  • ESP32 (micro-controller)
  • MPU6050 (Inertia measurement unit/Motion sensor)

Programming Languages and Libraries

  • C++ (ESP32 programming)
    • Arduino IDE (Transferring motion data to PC USB Port)
  • Python
    • Pytorch (Deep Learning)
    • Numpy, CV2, PIL (Data preprocessing)
    • Tensorboard (Visualization of Model Performance)
    • PySerial (Receiving incoming data from ESP32 through USB port)
    • Of course there are more...

Experimented Deep Learning Model

  • Convolution Neural Network (CNN): 92.2% test accurarcy
  • Artificial Neural Network (ANN): 76.0% test accurarcy
